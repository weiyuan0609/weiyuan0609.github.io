移动web基础知识

一.pixel 移动开发像素知识

px: css pixel 逻辑像素，浏览器使用的抽象单位

dp,pt： device independent pixels 设备无关像素

dpr: devicePixelRatio 设备像素比缩放

计算公式： 1px = (dpr)2 * dp

DPI: 打印机每英寸可以喷的墨汁点（印刷行业）

PPI： 屏幕每英寸的像素数量，即单位英寸内的像素密度

目前，在计算机显示设备惨呼描述上，两者意思表达的是一样的
